# Gerador de CPF em Python

Aplicação desenvolvida em Python que gera números de CPF válidos utilizando o algoritmo oficial de cálculo dos dígitos verificadores.

O programa cria automaticamente os nove primeiros dígitos e calcula os dois dígitos verificadores conforme as regras utilizadas para validação de CPFs.

---

## 🚀 Funcionalidades

- Geração automática dos 9 primeiros dígitos
- Cálculo do primeiro dígito verificador
- Cálculo do segundo dígito verificador
- Geração de um CPF válido completo

---

## 🧠 Conceitos de programação aplicados

- Lógica de programação

## ▶️ Como executar o projeto

1. Clone o repositório:

git clone https://github.com/seu-usuario/gerador-cpf-python.git

2. Entre na pasta do projeto:

cd gerador-cpf-python

3. Execute o programa:

python main.py
